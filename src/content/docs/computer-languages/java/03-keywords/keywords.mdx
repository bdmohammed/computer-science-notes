In Java, some identifiers are reserved to perform special functions.
These reserved identifiers are called **KEYWORDS** or **RESERVED WORDS**.

<Aside type="caution">
You **CANNOT** use them as identifiers (variable/class names).
</Aside>

**Why?** Because each keyword already has a fixed meaning in the compiler.

```java
int class = 10; // ❌ Compile-time error
```
